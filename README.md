## @react-navigation/bottom-tabs
Dùng bottom tab rất dễ dàng, ví dụ như có thể dễ dang thay đổi màu sắc của icon tab được chọn.
- initialRouteName="Home" màn hình sẽ được hiển thị đầu tiên là Home khi expo start.
